# Task Management System - Frontend

A modern, ClickUp-inspired Next.js frontend for the Data-Driven Task Management System.

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Axios (API Client)
- Recharts (Analytics Data Visualization)
- Lucide React (Icons)

## Setup
1. `cd frontend`
2. `npm install`
3. Create `.env.local` with `NEXT_PUBLIC_API_URL=http://localhost:5000/api`
4. `npm run dev`

## Features
- **Dashboard**: Real-time analytics and task overview.
- **Analytics**: Completion rates and creation trends visualization.
- **Task Management**: Advanced CRUD with pagination, filtering, and search.
- **Auth**: Secure JWT-based login and registration.
